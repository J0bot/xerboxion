# 🎮 Minecraft OS

> Minecraft Fabric prototype of Xerboxion

**Type:** Platform | **Status:** Beta | **Language:** Java

Minecraft Fabric implementation of Xerboxion concepts. Items: **BION**, **CUBION**, **XER**, **BOXION**, **XERBOXION**. Features a CC:Tweaked Lua kernel and fractal Cube Navigator. The CUBION item is the physical prototype of the cubion ploxion.

Mods: Applied Energistics 2, CC:Tweaked, Immersive Portals, MCEF.
