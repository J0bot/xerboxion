# 📱 Xerboxion OS

> Main OS platform — Rust/Flutter for Android

**Type:** Platform | **Status:** Alpha | **Language:** Rust / Flutter

The primary operating system implementation for Xerboxion. Provides the user-facing shell, HUD, and full integration with xerax (thought-control), xerboxion-core (runtime), and cubion (storage).
